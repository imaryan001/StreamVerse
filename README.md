# 🎬 StreamVerse  

A modern movie streaming application built for Android that allows users to explore and watch movies seamlessly.  
StreamVerse provides a clean UI, smooth browsing experience, and organized movie content for entertainment lovers.  

---

## ✨ Features  

- 🎥 Browse Movies Collection  
- 🔍 Search Movies  
- 📂 Categories (Action, Drama, Comedy, etc.)  
- ▶️ Watch Movies (Streaming UI)  
- ❤️ Add to Favorites  
- 🌙 Dark Mode Support  
- 📱 Clean and Responsive UI  

---

## 🛠️ Tech Stack  

- **Language:** Kotlin  
- **Platform:** Android  
- **Backend / API:** Movie API (e.g., TMDB)  
- **UI Design:** Material Design
